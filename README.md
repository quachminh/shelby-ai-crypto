# Shelby AI Crypto Assistant

This is an experimental crypto research assistant built to explore the Shelby SDK.

## Idea

The assistant helps users quickly analyze crypto projects by combining:

- On-chain data
- Social sentiment from X (Twitter)
- AI generated research summaries

## Example

User query:

Analyze Aptos ecosystem projects

Output:

- On-chain metrics
- Community sentiment
- Risk signals
- AI generated research summary

## Tech Stack

- Node.js
- Shelby SDK
- OpenAI
- Crypto APIs

## Goal

Build a lightweight AI tool that helps users research new crypto projects faster.
